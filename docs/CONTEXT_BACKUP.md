# Project Context & Progress Backup (CONTEXT_BACKUP)

Этот документ представляет собой **источник правды** о фактическом состоянии разработки проекта `QTab`. Он обновляется строго хронологически (самые новые изменения дописываются в самый конец файла) и содержит подробную информацию обо всех реализованных фичах, архитектурных решениях и интеграциях.

> [!IMPORTANT]
> **ПРАВИЛА ИЗМЕНЕНИЯ ЭТОГО ДОКУМЕНТА (ДЛЯ ИИ-АГЕНТОВ):**
> 1. Всегда соблюдайте строгий **хронологический порядок** (от старого к новому).
> 2. Каждое новое выполненное обновление (`## Update YYYY-MM-DD: ...`) должно **дописываться строго в самый конец файла**.
> 3. В начале файла должен оставаться только чистый вводный заголовок.
